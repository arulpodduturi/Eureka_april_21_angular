HOW to use CSS in an HTML file
=========================================

1.inline CSS 
    We use style attribute inside the element. Restrcited to one element
    <div style='color:red'> this is a div</div>
2. Internal CSS
    We use style tag
    restricted to one html page
3. External
    Write a CSS in separate .css file
    include that file in html file

HOW to use Bootstrap
===============================

ADD CDN link to the <head> of HTML file

How to write and execute Javascript code
=================================================
1. Browser (frontend development)

    a. create a JS file with some javascript content
    b. include that js file into html file
        <script src="link"></script>
    c. opent the HTML file in browser (alt+b)
2. NodeJS (backend Development)
    1. create a JS file with some js content
    2. run the js file with the below cmd
        node <filename>.js (make sure nodejs is installed)

Every js code that run on browser might not work in NodeJS

Most commonly used javascript functions
==================================================== 
console.log()       console: object, log : function
document.write()    document:object, write: function
window.alert()      window:object, alert: function

Note: while accessing window objects methods/functions, using window.function() is not mandatory
